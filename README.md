## Overview
Python Game Suite is a collection of interactive, easy-to-play games developed in Python. With a focus on fun and learning, these text-based games help users practice logical thinking, math skills, and general knowledge.

## Features

- Simple menu-driven interface for game selection
- Dynamic challenges: logic riddles, math puzzles, quizzes, word guessing
- Supports both single-player and two-player modes
- Real-time scoring and feedback for each game
- Replayable formatâ€”perfect for beginners and casual gamers

## Games Included

- Riddle Challenge
- Math Puzzle
- General Knowledge Quiz
- Rock-Paper-Scissors (AI Match)
- Rock-Paper-Scissors (Player vs Player)
- Word-Guessing Game

## Getting Started

1. Open `Python-Games.ipynb` in Jupyter Notebook or Google Colab.
2. Run the notebook cells and use the on-screen menu to choose and play games.

## Requirements

- Python 3.x
- Jupyter Notebook or Google Colab
